Where shall the new tab be opened (relative to the currently selected tab) when the toolbar button is clicked:
Default: Right
(To disable the toolbar button use Firefox' customization menu.)
Where shall the new tab be opened (relative to the currently selected tab) when a page's contextmenu is used:
Default: Disabled
Where shall the new tab be opened (relative to the tab clicked on) when a tab's contextmenu is used:
Default: Left
Where shall the new tab be opened (relative to the currently selected tab) when the tools menu entry is used:
Default: Right
Where shall the new tab be opened (relative to the currently selected tab) when the keyboard shortcut (Ctrl + Y) is used:
Default: Right
